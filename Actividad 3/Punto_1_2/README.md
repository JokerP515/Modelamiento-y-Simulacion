# Punto 1.2 - Visualización de Distribución con Histograma

## Requisitos Previos

Antes de ejecutar el código, asegúrese de tener instalado:

- **Python 3**: Puede verificar su versión con `python --version` o `python3 --version`
- **Librería matplotlib**: Instale con el siguiente comando:
  ```bash
  pip install matplotlib
  ```

## Instrucciones de Ejecución

### Usar datos de la carpeta Punto_1_1

Si desea usar datos recién generados:
1. Ejecute el generador en la carpeta `Actividad 2`
2. Copie el archivo `sequenceStandard.txt` generado a esta carpeta

### Generar el Histograma

Ejecute el script de Python para visualizar la distribución usando el siguiente comando:

```bash
python histogramDistribution.py
```
O bien usando cualquier IDE de su preferencia para ejecutar el archivo py.

Esto mostrará un histograma generado con matplotlib que representa la distribución de los números aleatorios.
